
The purpose of this project is to show how to create ETL programs using Scala3.

Scala can be used as scripting language.

Before running "scala create-etl-project.scala", it is needed to have installed:
==> Java vm 11 or later. https://adoptium.net/temurin/releases/
==> sbt, a build tool for Scala, Java and more. https://www.scala-sbt.org
==> scala3 https://www.scala-lang.org/download/

Now, after downloading ETLsWithScala latest release, it is possible to run:scala create-etl-project.scala
located insite projects directory.
